Question 2

\(fz(z) = (\frac{1}{2})e^{-|z|}\) can be rewritten as: \[ fz(z) = \begin{cases} (\frac{1}{2})e^{-|z|}, z \geq 0\\ (\frac{1}{2})e^{|z|}, z < 0 \end{cases} \]

Since \(X_1\) and \(X_2\) have exponential density, their PDF is:

\[ fx(x_1) = fx(x_2) = \begin{cases} e^{-x}, x \geq 0\\ 0, otherwise \end{cases} \]

So the combined density is \(^2e^{-(_1 + _2)}\)

If \(z = x_1 - x_2\) then \(x_1 = z + x_2\) \(=>\)

If \(z \geq 0\), then \(x_1 = (z + x_2) \geq 0\) and

\(fz(z) = \int_{z}^{} ^2e^{-(z + 2x_2)}, dx_2 = \frac{1}{2}e^{-z}\)

If \(z < 0\), then \(x_1 = (z + x_2) > 0\) and

\(fz(z) = \int_{0}^{} ^2e^{(z + 2x_2)}, dx_2 = \frac{1}{2}e^{z}\)

then \[ fz(z) = \begin{cases} (\frac{1}{2})e^{-z}, z \geq 0\\ (\frac{1}{2})e^{z}, z < 0 \end{cases} \]
